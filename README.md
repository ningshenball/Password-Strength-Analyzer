#  Password Strength Analyzer

A clean and professional **Streamlit web application** that analyzes password strength using smart detection rules. 

## Features

- Scores passwords out of **5.0** with decimal support
- Detects common weak passwords
- Identifies name-based patterns (including uncommon names)
- Detects sequential patterns and repeated characters
- Modern web interface with progress bar and clean UI
- Suggests stronger password alternatives

## How to Run Locally

```
git clone https://github.com/ningshenball/Password-Strength-Analyzer.git
cd Password-Strength-Analyzer
pip install streamlit
streamlit run app.py\
```
### Tech Stack

- Python 3
- Streamlit
- Regular Expressions (re)
